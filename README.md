# 🚗 Car Rental System

A console-based Car Rental Management System built with C++.

## 👩‍💻 Developer
- Yassmin Mansour

## 📋 Features
- Customer registration & login
- Admin panel for car management
- Add, remove & update cars
- Rent & return cars
- Cancel reservations
- Payment methods (Credit Card / Cash)
- Data saved to files

## 🛠️ Built With
- C++
- File Handling
- Structs & Arrays

## ▶️ How to Run
1. Open the project in Visual Studio
2. Build and run `Source.cpp`
